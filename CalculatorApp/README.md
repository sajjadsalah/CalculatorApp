# 📱 تطبيق الحاسبة - Calculator App

تطبيق حاسبة بسيط وأنيق لنظام iOS مبني بـ SwiftUI.

---

## 🚀 كيفية بناء IPA عبر GitHub Actions

### الخطوة 1: رفع المشروع على GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/اسمك/CalculatorApp.git
git push -u origin main
```

### الخطوة 2: تشغيل الـ Workflow
- افتح مستودعك على GitHub
- اضغط على تبويب **Actions**
- اضغط على **Build iOS IPA**
- اضغط **Run workflow**

### الخطوة 3: تحميل الـ IPA
- انتظر حتى ينتهي البناء (5-10 دقائق)
- اضغط على اسم الـ workflow الذي انتهى
- قم بتحميل **CalculatorApp-IPA** من قسم **Artifacts**

---

## ✍️ توقيع الـ IPA بشهادتك
بعد تحميل الـ IPA، وقّعه باستخدام:
- **Sideloadly** (مجاني)
- **AltStore**
- أي أداة توقيع تدعم شهادتك
